# Protocol
It's an bash script designed to resize and optimize JPG files, written in bash and designed for UNIX OS

I recommend 
```
chmod +x protocol.sh
```
or
```
chmod 777 protocol.sh
```
then use

```
./protocol.sh
```

##Requirements
It's recommended to have previouslly installed 

```
jpegoptim
```
```
convert
```
